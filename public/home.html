API estados de Venezuela

Esta API ofrece tres endpoints para obtener todas las entidades (estados, municipios y parroquias) que conforman el territorio de Venezuela

Endpoints

GET /all

Retorna todas las entidades

https://api-venezuela.edixonserrano.com/all

GET /state/:estado

Retorna la informacion de un estado especifico pasando el nombre de este como parámetro. En caso de que el nombre del estado contenga espacios en el nombre estos deben ser eliminados, por ejemplo: Nueva Esparta sera NuevaEsparta. Minúsculas y mayúsculas no tienen relevancia.

https://api-venezuela.edixonserrano.com/state/{estado}

GET /state/:estado/:municipio

Retorna la informacion de un municipio especifico pasando el nombre de este como parámetro, antecedido del nombre del estado. En caso de que el nombre del municipio contenga espacios en el nombre estos deben ser eliminados, por ejemplo: Carlos Arvelo sera CarlosArvelo. Minúsculas y mayúsculas no tienen relevancia.

https://api-venezuela.edixonserrano.com/state/{estado}/{municipio}
